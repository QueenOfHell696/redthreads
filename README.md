# Red Threads

## Самый простой способ выложить сайт в сеть (без установки чего-либо)

1. Зайди на https://app.netlify.com/drop
2. Открой папку `dist` (см. ниже, как её получить) и перетащи её прямо в окно браузера на этом сайте
3. Netlify сама выдаст ссылку вида `https://random-name-123.netlify.app` — это и есть твой сайт

Но папку `dist` сначала нужно собрать — это делается один раз на компьютере:

### Как собрать dist

1. Установи Node.js (если ещё нет): https://nodejs.org — просто скачать и поставить, как обычную программу
2. Открой терминал (командную строку) в папке проекта
3. Выполни по очереди:

```
npm install
npm run build
```

4. Появится папка `dist` — вот её и тащишь на Netlify Drop

## Если хочешь через GitHub (чтобы обновлять сайт позже одной кнопкой)

1. Залей эту папку в репозиторий на GitHub
2. Зайди на https://vercel.com или https://app.netlify.com, подключи GitHub-репозиторий
3. Настройки сборки подхватятся сами (Vite-проект): команда сборки `npm run build`, папка `dist`
4. Каждый раз, когда меняешь код и заливаешь на GitHub — сайт обновляется сам

## Важно про данные

Доски хранятся в браузере (localStorage) — то есть только на том устройстве и в том браузере, где ты их создавал. Если открыть сайт с телефона или из другого браузера — досок там не будет, это будет чистый лист.
